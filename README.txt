Petit projet de jeu vidéo de "Click and destroy" fait en pur Javascript dans le but de s'amuser, et par curiosité, à partir d'un tuto sur internet.

Il est possible de gérer la difficulté à partir du fichier script.js qui contient tout le code Javascript, les variables au début du fichier représentent le nombre de clicks restants avant la fin de la partie (la partie se termine une fois les munitions épuisées), et le nombre maximum d'ennemis présents dans le jeu avant Game over. 
On peut donc tout à fait augmenter cette limite pour rendre le jeu plus simple ou au contraire la réduire pour un jeu plus rapide.

Le style est codé avec le langage préprocesseur CSS Sass (SCSS) et se trouve dans le fichier main.scss .

Je me suis amusé à recréer la célèbre map "Dust 2" de Counter Strike où j'ai passé une partie de mon enfance en tant que petit thème amusant.

Bon jeu ! :D
